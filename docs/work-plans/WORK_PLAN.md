# QuantLab 工作计划

更新日期：2026-06-24

本文档用于跟踪当前本地 A 股量化回测平台的开发状态，包含已完成事项、正在处理事项和未完成路线图。

## 已完成

| 模块 | 状态 | 说明 |
|---|---:|---|
| 本地 Web 架构 | ✅ | FastAPI 后端 + React/Vite 前端，可在本机运行。 |
| A 股日线回测 MVP | ✅ | 支持日线频率，`frequency` 字段保留分钟线扩展入口。 |
| 可视化策略配置 | ✅ | 支持均线/价格均线/RSI 条件组合，买卖条件可视化编辑。 |
| 回测核心规则 | ✅ | 收盘信号、下一交易日开盘成交、T+1、整手交易、交易成本、滑点、最大仓位。 |
| 涨跌停/停牌约束 | ✅ | 支持涨停不买、跌停不卖、停牌不交易；按分比较价格边界。 |
| 数据集快照 | ✅ | CSV/AkShare 数据固化为 SHA-256 指纹快照，支持去重、预览和回测追溯。 |
| AkShare 数据接入 | ✅ | 支持沪深日线、沪深300基准、交易日历、东方财富失败后新浪回退。 |
| CSV 导入 | ✅ | 支持 10MB CSV 导入、OHLCV 校验、交易日过滤和持久化。 |
| 回测任务持久化 | ✅ | SQLite 保存任务、配置、进度、取消状态和结果，重启后可查看历史。 |
| 项目/策略版本 | ✅ | 支持默认项目、策略保存、不可变版本和回测记录关联。 |
| 前端数据中心 | ✅ | 支持演示数据、CSV 快照、AkShare 快照三路切换。 |
| 前端模块化 | ✅ | 已将单文件 `App.jsx` 拆为配置、格式化、通用控件、抽屉/弹窗和仪表盘展示模块，主组件只保留状态与业务动作。 |
| 证券主数据/状态表 | ✅ | SQLite 增加 `securities` 与 `security_daily_status`，导入 CSV/AkShare 快照时抽取上市日、交易所、板块、ST/停牌/涨跌停日状态，回测创建前校验上市/退市窗口。 |
| 证券主数据来源增强 | ✅ | 增加 AkShare 沪深北证券主数据同步，支持名称、交易所、板块、上市日、退市日、行业、总股本和流通股本；提供 `/api/securities/sync/akshare`。 |
| 复权因子与公司行为校验 | ✅ | 数据快照标准化 `adjust_factor`、`adjusted_close`、`corporate_action`、`adjustment_anomaly`，并持久化数据集复权质量检查；回测结果披露当前“未复权撮合/信号”口径。 |
| 复权信号口径切换 | ✅ | `signal_price_mode` 支持未复权/复权收盘价计算策略信号；撮合、估值、现金和费用仍固定使用未复权价格，前端设置中可切换。 |
| 涨跌停规则版本化（新股豁免） | ✅ | 快照增加 `limit_rate`、`limit_exempt`、`limit_reason`；支持北交所上市首日、科创/创业/注册制主板前 5 个交易日不设涨跌幅限制，撮合引擎会跳过豁免日涨跌停拒单。 |
| 涨跌停规则版本化（老主板新股） | ✅ | 注册制前沪深主板新股上市首日按 44%/-36% 特殊价格边界生成 `limit_up/limit_down`，并通过 `limit_reason` 披露。 |
| 行业/板块历史与长期停牌 | ✅ | 增加 `security_industry_history`，AkShare 主数据和数据集导入会记录行业/板块版本；日状态增加连续停牌天数和长期停牌标记。 |
| 行业历史 CSV 数据源 | ✅ | 新增行业历史 CSV 导入接口和数据中心入口，支持 `symbol, valid_from, industry, board` 版本化导入。 |
| 前端股票池性能优化 | ✅ | 数据抽屉增加股票搜索，股票池渲染限制为前 200 条匹配结果，避免同步全市场证券后一次渲染数千按钮。 |
| 报告导出与交易分页 | ✅ | 增加 Markdown 回测报告下载、交易记录分页 API，前端“导出报告”按钮已接入当前回测任务。 |
| HTML/PDF 报告导出 | ✅ | 新增 HTML 在线报告和 PDF 下载接口，HTML 内置净值 SVG、指标卡、质量检查、拒单统计和交易表；前端提供 HTML/PDF 两个导出入口。 |
| 报告批注与对比图增强 | ✅ | 回测请求支持研究备注，Markdown/HTML/PDF 报告会披露批注；最近回测对比增加迷你收益条。 |
| 前端稳定性与懒加载 | ✅ | 增加全局错误边界、启动/回测加载提示，并将 Recharts 图表拆成懒加载 chunk，降低主包体积。 |
| 结果分析分页与拒单统计 | ✅ | 交易表接入分页 API，支持买入/卖出方向筛选和翻页；图表区展示涨停、跌停、资金不足等订单拒绝统计。 |
| 多次回测对比 | ✅ | 新增最近完成回测对比 API，前端展示累计收益、年化收益、回撤、夏普和交易次数对比表。 |
| 策略风控参数 | ✅ | 回测请求支持单标的仓位上限、调仓周期、止盈和止损；交易记录披露买卖原因，前端设置面板已接入。 |
| 股票池过滤器与扩展指标 | ✅ | 回测请求支持排除 ST、最少上市天数、20 日均成交额过滤；策略条件新增 MACD 和布林带，前端可视化规则节点可切换指标与条件。 |
| 审查报告 2026-06-23 修复 | ✅ | 修复前端 NaN/undefined 崩溃、请求错误静默、Abort 竞态、轮询退避、中文无障碍、任务取消竞态、损坏 JSON 降级、真实日期校验和复权公司行为日止损误触发。 |
| 工程卫生与导入性能 | ✅ | Vite 构建依赖移至 devDependencies、入口 root 节点增加防御、数据库迁移字段白名单、数据集持久化复用已标准化 DataFrame 降低重复处理。 |
| 数据质量前端展示 | ✅ | 数据中心选择固定快照后展示复权因子覆盖、公司行为标记和复权连续性等质量检查结果。 |
